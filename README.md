# Rivalo

Rivalo è una piattaforma sportiva sociale per calcetto, padel e tennis.

## Funzioni MVP previste

- Login utenti
- Gruppi sportivi
- Inviti amici
- Richieste partita
- Calendario
- Ranking
- RivalScore
- Card giocatore
- Badge e gamification

## Avvio locale

```bash
npm install
npm run dev
```

## Firebase

Copia `.env.example` in `.env.local` e inserisci le chiavi Firebase.
